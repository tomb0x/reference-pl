= tan(rad)


// POCZĄTEK SEKCJI OPISOWEJ
[#overview]
--

[float]
=== Opis
Oblicza tangens kąta (w radianach). Wynik będzie między ujemną a dodatnią nieskończonością.
[%hardbreaks]


[float]
=== Składnia
`tan(rad)`


[float]
=== Parametry
`rad`: Kąt w radianach. Dozwolone typy danych: `float`.


[float]
=== Wartości zwracane
Tangens kąta. Typ danych: `double`.

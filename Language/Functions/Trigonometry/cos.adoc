= cos(rad)


// POCZĄTEK SEKCJI OPISOWEJ
[#overview]
--

[float]
=== Opis
Oblicza kosinus kąta (w radianach). Wynik będzie pomiędzy -1 a 1.
[%hardbreaks]


[float]
=== Składnia
`cos(rad)`


[float]
=== Parametry
`rad`: Kąt w radianach. Dozwolone typy danych: `float`.


[float]
=== Wartości zwracane
Kosinus kąta. Typ danych: `double`.

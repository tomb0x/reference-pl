= highByte()


// POCZĄTEK SEKCJI OPISOWEJ
[#overview]
--

[float]
=== Opis
Wyodrębnia bajt bardziej znaczący (skrajnie lewy) ze słowa (lub drugi mniej znaczący dłuższego typu danych).
[%hardbreaks]


[float]
=== Składnia
`highByte(x)`


[float]
=== Parametry
`x`: wartość dowolnego typu


[float]
=== Wartości zwracane
Typ danych: `byte`.

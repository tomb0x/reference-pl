= word

// POCZĄTEK SEKCJI OPISOWEJ
[#overview]
--

[float]
=== Opis
Typ word (słowo) przechowuje 16-bitową liczbę bez znaku (z zakresu od 0 do 65535).
[%hardbreaks]


[float]
=== Składnia
`word zm = wart;`


[float]
=== Parametry
`zm`: nazwa zmiennej. +
`wart`: wartość przypisana do zmiennej.

--

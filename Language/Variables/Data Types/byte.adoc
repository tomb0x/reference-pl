= byte

// POCZĄTEK SEKCJI OPISOWEJ
[#overview]
--

[float]
=== Opis
Typ byte (bajt) przechowuje 8-bitową liczbę bez znaku (z zakresu od 0 do 255).
[%hardbreaks]


[float]
=== Składnia
`byte zm = wart;`


[float]
=== Parametry
`zm`: nazwa zmiennej. +
`wart`: wartość przypisana do zmiennej.

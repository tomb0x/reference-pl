= length()


// POCZĄTEK SEKCJI OPISOWEJ
[#overview]
--

[float]
=== Opis
Podaje długość ciągu, wyrażoną w znakach. (Zauważ, że nie obejmuje to końcowego znaku zerowego)

[%hardbreaks]


[float]
=== Składnia
`mójŁańcuch.length()`


[float]
=== Parametry
`mójŁańcuch`: zmienna typu `String`.


[float]
=== Wartości zwracane
Długość ciągu w znakach.

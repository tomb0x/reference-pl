= toUpperCase()


// POCZĄTEK SEKCJI OPISOWEJ
[#overview]
--

[float]
=== Opis
Dostarcza wersję ciągu pisaną wielkimi literami. Począwszy od wersji 1.0, toUpperCase() modyfikuje ciąg, zamiast zwracać nowy.

[%hardbreaks]


[float]
=== Składnia
`mójŁańcuch.toUpperCase()`


[float]
=== Parametry
`mójŁańcuch`: zmienna typu `String`.


[float]
=== Wartości zwracane
Nic

= charAt()


// POCZĄTEK SEKCJI OPISOWEJ
[#overview]
--

[float]
=== Opis
Dostęp do konkretnego znaku w ciągu znaków.

[%hardbreaks]


[float]
=== Składnia
`mójNapis.charAt(n)`


[float]
=== Parametry
`mójNapis`: zmienna typu `String`. +
`n`: zmienna. Dozwolone typy danych: `unsigned int`.


[float]
=== Wartości zwracane
Znak na pozycji n w ciągu znaków.
